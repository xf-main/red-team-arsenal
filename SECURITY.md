# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing the maintainer or opening a confidential issue.

**Please do not** disclose security vulnerabilities publicly until they have been addressed.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Security Best Practices

When using this project:

1. Always use the latest version
2. Review code before execution
3. Use in authorized environments only
4. Follow security guidelines in documentation
5. Keep dependencies updated

## Responsible Disclosure

We appreciate responsible disclosure of security issues. We will:

1. Acknowledge receipt of your report
2. Investigate and validate the issue
3. Work on a fix
4. Credit you for the discovery (if desired)

Thank you for helping keep this project secure!
